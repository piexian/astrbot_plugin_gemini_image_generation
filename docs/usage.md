# 使用指南

## 命令列表

| 命令 | 说明 | 示例 |
|------|------|------|
| `/生图` | 纯文本生成 | `/生图 一只可爱的橙色小猫` |
| `/改图` | 基于参考图修改 | 发送图片 + `/改图 把头发改成红色` |
| `/换风格` | 风格转换 | 发送图片 + `/换风格 水彩` |
| `/快速 头像` | 头像模式 | `/快速 头像 商务风格` |
| `/快速 海报` | 海报模式 | `/快速 海报 赛博朋克` |
| `/快速 壁纸` | 壁纸模式 | `/快速 壁纸 未来城市` |
| `/快速 卡片` | 卡片模式 | `/快速 卡片 商务名片` |
| `/快速 手机` | 手机壁纸 | `/快速 手机 极简风格` |
| `/快速 手办化` | 手办效果 | `/快速 手办化 [1/2] 动漫角色` |

## 智能头像功能

启用 `auto_avatar_reference` 后，插件会根据自然语言自动获取头像作为参考图：

| 触发方式 | 行为 |
|----------|------|
| `按照我` / `根据我` / `基于我` / `我的头像` | 获取发言人头像 |
| `@某人` + 生图/改图命令 | 获取被 @ 用户头像 |

优先级：`@ 指定用户 > 发言者自己`。

## 快速模式

| 模式 | 默认分辨率 | 默认比例 | 说明 |
|------|------------|----------|------|
| 头像 | `1K` | `1:1` | 个人头像 |
| 海报 | `2K` | `16:9` | 宣传海报 |
| 壁纸 | `4K` | `16:9` | 高清壁纸 |
| 卡片 | `1K` | `3:2` | 名片卡片 |
| 手机 | `2K` | `9:16` | 手机壁纸 |
| 手办化 | `2K` | `3:2` | 树脂手办效果，参数 `1` 为 PVC，`2` 为 GK |
| 表情包 | `4K` | `16:9` | Q 版 LINE 风格表情包 |

快速模式的默认分辨率和比例可通过 `quick_mode_settings` 覆盖。

## 切图指令

- 智能切分：`/切图`
- 手动网格：`/切图 4 4`、`/切图 44`、`/切图4x4`
- 主体吸附：指令包含 `吸附` 时启用主体+附件吸附算法

切图流程：

```text
手动网格 > AI 识别（需配置 vision_provider_id）> 智能网格 > 主体吸附兜底
```

表情包生成质量和模型输出强相关。如果模型没有生成稳定网格，自动切图可能切到主体、文字或空白区域。遇到这类情况优先尝试手动网格，或使用专门切图工具二次处理。

## LLM 工具集成

插件注册 `gemini_image_generation` 工具后，模型可以用自然语言触发生图：

- `帮我画一只可爱的小猫`
- `把我的头像改成动漫风格`
- `基于这张图生成一个海报`

### 前台/后台混合模式

LLM 工具会根据当前 `tool_call_timeout` 和 `llm_tool_timeout_reserve_percent` 计算前台等待窗口：

- 图片在窗口内生成完成时，以 `CallToolResult` 结构化返回图片。
- 超过等待窗口时，自动切到后台继续生成，完成后通过 `event.send()` 直接发送给用户。

代理模式下，前台和后台都会通过代理下载远程图片，保证链路一致。

### 参数动态切换

使用 `openai_images` 且启用 `size_mode=custom` 时：

- LLM 工具仅接受 `size` 参数，格式 `WxH`。
- 不再接受 `resolution` 和 `aspect_ratio`。
- 未显式传入 `size` 时，使用配置中的 `openai_images.custom_size`。

其他供应商或非 custom 模式继续使用 `resolution` / `aspect_ratio`。

### 参数校验

LLM 传入非法 `resolution`、`aspect_ratio` 或 `size` 时，工具会直接返回错误并要求模型修正后重试，不会静默回退默认值。

### Gemini thought_signature 处理

插件会把 Gemini 返回的 `thought_signature` 视为仅限协议层使用的 opaque 元数据，不会拼进 Tool 文本结果或用户可见内容。这可以避免部分 Gemini / NewAPI 网关把超大签名重新塞回上下文，导致 `413` 或输入 token 超限。

## 论坛发帖模式

当用户明确要求将图片发到论坛或 AstrBook 时，LLM 会设置 `for_forum=true`。此时工具会同步等待生成完成并返回图片路径或 URL，LLM 可继续调用 `upload_image` 上传图床后自动发帖。

## 表情包切分算法

SmartMemeSplitter v4 包含：

- 颜色边缘突变分析：彩色形态学梯度 + OTSU 自适应阈值
- 能量图分析：基于 Sobel 算子的多通道能量计算
- 投影分析：水平/垂直投影检测网格边界
- 网格候选微调：精细调整以寻找更清晰的分隔线

主体吸附算法会尝试去除白底、提取前景掩码、识别主体与附件，并把裁剪结果转为透明背景。该路径适合作为兜底，不保证能处理所有模型输出。
